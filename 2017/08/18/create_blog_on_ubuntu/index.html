Ubuntu14.04环境下利用github.io+hexo搭载个人博客

发表于 2017-08-18

1.首先注册github账号并登录, 然后创建一个新的仓库(repository)

2.安装git

  • 在终端中输入:
    sudo apt-get install git-core

  • 检查是否安装成功:
    git –version

3.配置node.js环境

  • 在安装node.js之前先安装g++和curl
  • 在终端中输入:
    sudo apt-get install g++
    sudo apt-get install curl
  • 配置nvm(node版本管理器, 可以让你的环境部署多个版本的node), 在终端中输入:
    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

其它方式请参考:nvm github

  • 安装node.js:
    nvm install 4
  • 检测node.js是否安装成功:
    nvm ls


笔者开始安装了node.js 0.10, 后面使用hexo时各种坑
Tips:设置默认的node.js版本,在终端中输入: nvm alias default v4.8.4

4.初始化hexo

  • 新建一个空目录
    mkdir blog
    cd blog
  • 初始化
    hexo init
  • 本地部署
    hexo server
  • 在浏览器中输入http://localhost:4000/. 至此,一个基本的blog就搭建好了

  • 部署
    终端中输入命令:
    hexo clean && hexo g
    hexo d

6.hexo的常用命令

  • 新建文章
    hexo new “file_name”
  • 清楚网页缓存
    hexo clean
  • 生成静态网页
    hexo g
  • 本地部署
    hexo s/server
  • 命令可以连用, 比如新写好一篇博文后, 输入
    hexo clean && hexo g && hexo s
    即可本地部署查看
